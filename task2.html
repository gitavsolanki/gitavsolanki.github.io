Task 3 – Creating and Editing a Document

This task checks that you can create, edit, and format a document.

What you need

  • A word processing application (for example: Microsoft Word)
  • The text and image provided by your tutor

Steps to complete the task

  1. Open a word processing application (for example: Microsoft Word).
  2. Create a new blank document.
  3. Copy the text provided into your document.
  4. Format the text so it is clear and easy to read (for example: change font size, use bold for headings).
  5. Insert the image and align it in the centre below the text.
  6. Save the document as: YourInitials_Task3
